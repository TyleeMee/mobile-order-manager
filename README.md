# mobile-order-manager

## 概要
モバイルオーダー 店舗管理画面

## 使用技術
- フロントエンド: Next.js (React), TypeScript, Tailwind CSS
- バックエンド: Node.js, Express
- データベース: PostgreSQL
- インフラ: AWS (EC2, S3, RDS)
- デプロイ: Vercel
